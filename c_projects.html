Hassan's C and MIPS Assembly Projects (from CMSC216)

Fall 2023 Semester - Course Instructor: Larry Herman

Check Lines
 I practiced the use of standard library I/O functions and event trees to check for lines of code that violate style guidelines for any input file.
Blueberry Crisp
 Practiced bit masking that converts machine language into assembly instructions for a fictional processor and vice versa.
Makefiles
 Created makefiles that used proper dependencies, macros, and style.
Walrus
 I familiarized myself with C structs, unions, and pointer arithmetic.
Dynamic Memory Allocation
 I practiced dynamic memory allocation with structs and data, prevented any sort of memory leaks, and C string library functions.
Priority Queues and Lists
 Continued to use structs to create linked data structures that simulated priority queues and lists of priority queues while using dynamic memory allocation and preventing memory leaks.
Process Control
 Practiced process control concepts like forking, standard I/O, input/output redirection, and exec functions.
Submit Server
 Created a functional submit server from scratch which uses process control, piping, I/O redirection, and makefiles to compile and test student code.
Assembly
 Converted C programs into MIPS assembly instructions.
Threads
 Practiced concurrency with PThreads.
