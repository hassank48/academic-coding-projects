Hassan's Java Projects (from CMSC132)

Some GUI's and advanced software were not my work, but the work of my course instructor Fawzi Emad. However, all concepts listed below were my own code and did not use outside assistance.

Blackjack Simulator

I practiced the use of ArrayLists, enumerations, logic, classes, and interfaces in order to make a functional Blackjack simulator.

BasicLinkedList

I used Node structures, parametrized types, and iterators to make my own functional linked list.

MyHashSet

Used an array of Node linked lists to create my own Hash Set/Table.

Minimum Snippet

Using iterators, I replicated a search engine feature that finds a sequence of words/letters across pages/documents with no regard for order.

Binary Search Tree

Used interfaces, objects, and event trees in order to create a functional binary search tree.

Maze Graph

Used hashmaps and search algorithms (Depth and Breadth First Searches and Dijkstra's) to create weighted graphs and solve intricate mazes and find their optimal weighted paths.

HeavyBag

Used hashmaps to create a data structure that can hold large amounts of duplicate objects optimizing performance.
